== Frequently Asked Questions ==

= What does Innozilla CF7 Skins do? =

Innozilla CF7 Skins is an add-on plugin for Contact Form 7. It extends the functionality of Contact Form 7 by just tweaking the settings that can be found in the "Skin/Style - ICF7" tab.

= Do I need to have coding skills to use Innozilla CF7 Skins? =

Absolutely not. You can create and manage Contact Form 7 forms without any coding knowledge. (Very Simple form settings)
